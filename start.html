Current Project: Game Jam Projects

This starter area generator was something fun I made in high school. The basis for the code originated from something my dad put together which used dice rolls to generate areas. I started to use it for more than just starter areas though, and ended up programming it into what it is now so I could do it without rolling dice. A recent addition is the part that includes area "Strengths" with a dice roll as well. This was a concept that essentially helped to add more flavor to locations made by this generator, and I had a lot of fun making it.
